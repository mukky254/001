How It Works

Simple three-step process

1

Lecturer Generates QR Code

Lecturer creates a unique QR code for each class with specified duration.

2

Student Scans QR Code

Students use their mobile devices to scan the QR code during class.

3

Attendance Recorded

System automatically records attendance with timestamp and stores in database.
